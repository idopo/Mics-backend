**Deduplication (REQUIRED):** GET /api/toolkits returns multiple rows per toolkit name (one per hw_hash variant). The page must show one card per unique name. Deduplicate using a Map keyed on toolkit name, keeping the entry with the latest updated_at:

```typescript
function deduplicateToolkits(toolkits: ToolkitRead[]): ToolkitRead[] {
  const map = new Map<string, ToolkitRead>()
  for (const tk of toolkits) {
    const existing = map.get(tk.name)
    if (!existing || tk.updated_at > existing.updated_at) {
      map.set(tk.name, tk)
    }
  }
  return Array.from(map.values())
}
```

Call this inside the component after the query resolves: `const toolkitCards = deduplicateToolkits(toolkits ?? [])`
